Amro Zaid - Mobile Developer Portfolio (Lab 2.1)

Overview

This repository hosts my personal portfolio website, refactored and updated from Lab 1. The primary goal of this revision was to build a modern, professional, and responsive website using Bootstrap 5 to showcase my academic background and unique technical skills.

The site is fully responsive and optimized for viewing across all devices.
